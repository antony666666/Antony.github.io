Kubernetes Grafana

Kubernetes Grafana

部署 Grafana on K8S

環境準備
CNT.2022.v4 K8S 叢集建置完成
以下 K8S 公設完成部署

  • MetalLB
  • Ingress NGINX Controller
  • Local Path Provisioner

在 Windows 系統的 cmd 視窗登入 m1 主機

部署 Grafana

登入 Grafana

打開瀏覽器，網址 http://gf.k8s.org

帳號：grafana
密碼：Grafana12345

新增 Data Sources

點 ‘Configuration’ (左側齒輪)
點 ‘Add data source’ (中間藍色框)
點 ‘Mysql’
MySQL Connection

  • Host: mariadb.landlord.svc:3306
    • Database: landlord
    • User: landlord
    • Password: mymariadblandlord

新增 Dashboard

點 ‘Create’ (左側 +)
點 ‘Refresh dashboard’ (右上倒三角)
點 ‘5s’
點 ‘Save dashboard’
取名為 Building

新增 Panel

版面說明

設定時鐘

Panel 選單選取 ‘Clock’
Panel 設定

  • Panel options
    • Title: Time
    • Time Format
  • Clock Type: 24 Hour
    • Font size: 20px
    • Font size: Bold
  • Date Options
    • Font size: 16px

設計 History 表格

Panel 選單選 ‘Table’
Panel 設定
Panel options
Title: History
Database Query
Edit SQL
SELECT * FROM history
Format as: Table
